# Gold Stars

The seven features that make ralph work. Don't cut these.

## 1. Re-Read Loop

Pi gets the full plan every task. Prevents drift.

## 2. One Task Focus

Prompt says: "Do this task. Do NOT proceed to next."

## 3. Build Gate

`ralph done` runs build. Fail = fix first. Pass = proceed.

## 4. Stuck → Help

`ralph stuck` is easier than hacking. Oracle or codex review.

## 5. THINK.ALIGN.ACT

Every prompt includes:
```
THINK: Understand before acting
ALIGN: Get approval on significant changes
ACT:   Only after thinking and aligning
```

## 6. Living Notes

`ralph note "reason"` captures decisions with timestamp.

## 7. Progress Visible

```
Progress: ████████░░░░ 4/7 (57%)
```

Always know where you are.

---

## The Key Insight

> Help must be easier than hacking.

Agents hack because it's faster than asking. `ralph stuck` inverts that.
